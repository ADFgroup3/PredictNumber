色々内容書き換えて、数字の画像ファイルを読み込んでその数字が何かを予測し表示するようにはできた。
Pythonとhtmlを使ってローカル上でアプリケーションにするところまではできたんだけど、
Azureで色々やる所からよくわかんなかったんでこっからはゆーとくんに丸投げします。よろしく。
